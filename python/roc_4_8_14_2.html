def draw_graph_with_rates(net, s0_in):
    G=nx.DiGraph()
    G.add_nodes_from(s0_in.keys())
    edges = []
    for node in net.node_names:
        for onode in net.node_dict[node].sends_to:
            edges.append((node, onode))
    G.add_edges_from(edges)
    theta = np.arange(0, 720, len(s0_in.keys()))
    pos = {}
    for i in range(len(s0_in.keys())):
        pos[s0_in.keys()[i]] = (np.cos(i), np.sin(i))
    pos['E'] = (0,0)
    try :
        pos['E1'] = (2,0)
    except KeyError:
        pass
    val_map = {}
    for (key, val) in s0_in.iteritems():
        if val=='infected':
            val_map[key] ='r'
        else: 
            val_map[key] = 'g'
    values = [val_map.get(node, 0.25) for node in G.nodes()]
    nx.draw(G, pos=pos, node_color = values)

Empirical Error bars¶

Since we have 8000 of each datasets, we can see the variation of the ROC due to low number of samples. To generate the "confidence interval", I first generated 8000 datasets with attackers and 8000 datasets without attacker. Then I sampled 800 of each case 100 times, where each of the 100 times would produce a new ROC curve. The dark lines in the middle of the shaded region represent the median of all of the ROC's for that particular value along the x axis (false positives). The shaded regien represents the regiion where for any point on the x axis, 95% of the ROC's had points in the shaded region.

To generate the error "confidence interval" above, I first generated 8000 datasets with attackers and 8000 datasets without attacker. Then I sampled 800 of each case 100 times, where each of the 100 times would produce a new ROC curve. The dark lines in the middle of the shaded region represent the median of all of the ROC's for that particular value along the x axis (false positives). The shaded regien represents the regiion where for any point on the x axis, 95% of the ROC's had points in the shaded region.
